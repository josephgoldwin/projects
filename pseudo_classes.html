Negation & Filtering

No. Pseudo_class Description Example Output
1 :not(selector) Selects elements that do not match a given selector. <input type="text" class="not">
2 :not(:first-child) Selects elements that are not the first child of their parent. <input type="text" class="not-first-child">
3 :not(:last-child) Selects elements that are not the last child of their parent. <input type="text" class="not-last-child">
4 :not(:only-child) Selects elements that are not the only child of their parent. <input type="text" class="not-only-child">
5 :is(selector) Matches elements that match any of the selectors in the list. <input type="text" class="is">
6 :where(selector) Matches elements like :is() but with zero specificity. <input type="text" class="where">
7 :has(selector) Selects elements that contain elements matching the selector. <input type="text" class="has">
