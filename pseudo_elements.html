Pseudo-Elements

S.No. Selector Description Example (HTML) Result (CSS)
1 ::after Inserts content after the content of an element. <p>Hello</p> p::after { content: " World"; }
2 ::before Inserts content before the content of an element. <p>Hello</p> p::before { content: "Hi! "; }
3 ::first-letter Styles the first letter of the first line of a block element. <p>Hello world</p> p::first-letter { font-size: 2em; }
4 ::first-line Styles the first line of a block element. <p>Hello world</p> p::first-line { color: red; }
5 ::placeholder Styles placeholder text in an input or textarea. <input placeholder="Enter name"> input::placeholder { color: gray; }
6 ::selection Styles text selected by the user. <p>Select this text</p> ::selection { background: yellow; }
7 ::marker Styles the bullet or numbering of a list item. <ul><li>Item</li></ul> li::marker { color: blue; }
8 ::backdrop Styles the backdrop of elements in full-screen mode. <video controls></video> ::backdrop { background: rgba(0,0,0,0.8); }
9 ::file-selector-button Styles the button of an <input type="file"> element. <input type="file"> input::file-selector-button { background: green; }
10 ::cue Styles WebVTT captions and cues for media. <video></video> ::cue { color: yellow; }
11 ::cue-region Styles the entire area where WebVTT cues are displayed. <video></video> ::cue-region { background: black; }
12 ::slotted(selector) Styles elements assigned to a slot in Shadow DOM. <slot></slot> ::slotted(span) { color: red; }
13 ::part(name) Styles elements inside Shadow DOM that have a part attribute. <div part="button">Click</div> ::part(button) { background: red; }
14 ::grammar-error Highlights grammar errors detected by the browser. <p>He go to school</p> p::grammar-error { text-decoration: underline wavy green; }
15 ::spelling-error Highlights spelling errors detected by the browser. <p>Helo world</p> p::spelling-error { text-decoration: underline wavy red; }
16 ::target-text Styles the text target of a URL fragment. <p id="text">Target me</p> ::target-text { background: yellow; }
17 ::view-transition Styles elements during a view transition animation. <div>Page Content</div> ::view-transition { opacity: 0.5; }
