Global Attributes

No. Attribute Description
1 accesskey Defines a shortcut key to access an element
2 class Defines the class of an element
3 contenteditable Specifies whether the content of an element is editable or not
4 dir Specifies the text direction for the content of an element
5 draggable Specifies whether an element is draggable or not
6 hidden Specifies that an element is not to be rendered
7 id Specifies a unique id for an element
8 lang Specifies the language of the element's content
9 spellcheck Specifies whether the element should have its spelling and grammar checked or not
10 style Specifies an inline CSS style for an element
11 tabindex Specifies the tabbing order of an element
12 title Specifies extra information about an element
13 translate Specifies whether the content of an element should be translated or not
14 data-* Specifies custom data attributes for an element
15 enterkeyhint Specifies the enter key hint for an element
16 inputmode Specifies the input mode for an element
17 insert Specifies the insertion mode for an element
18 popover Specifies the popover for an element
