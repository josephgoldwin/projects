7 onunload script Executed when a page is unloaded

9 onoffline script Executed when the user goes offline
10 ononline script Executed when the user goes online
11 onpagehide script Executed when a page is hidden
12 onpageshow script Executed when a page is shown
13 onpopstate script Executed when the history changes
14 onresize script Executed when the browser window is resized
15 onstorage script Executed when the Web Storage area is updated
16 onunload script Executed when a page is unloaded

Form Event Attributes

1 onblur script Executed when an element loses focus
2 onchange script Executed when an element's value changes
3 oncontextmenu script Executed when the user right-clicks an element
4 onfocus script Executed when an element gets focus
5 oninput script Executed when an element is changed
6 oninvalid script Executed when an element is invalid
7 onreset script Executed when a form is reset
8 onsearch script Executed when the user searches for something
9 onselect script Executed when an element is selected
10 onsubmit script Executed when a form is submitted

Keyboard Event Attributes

1 onkeydown script Executed when a key is pressed down
2 onkeypress script Executed when a key is pressed
3 onkeyup script Executed when a key is released

Mouse Event Attributes

1 onclick script Executed when an element is clicked
2 ondblclick script Executed when an element is double-clicked
3 onmousedown script Executed when the user presses a mouse button over an element
4 onmousemove script Executed when the user moves the mouse over an element
5 onmouseout script Executed when the user moves the mouse out of an element
6 onmouseover script Executed when the user moves the mouse over an element
7 onmouseup script Executed when the user releases a mouse button over an element
8 onmousewheel script Executed when the user scrolls the mouse wheel over an element
9 onwheel script Executed when the user scrolls the mouse wheel over an element

Drag Event Attributes

1 ondrag script Executed repeatedly while an element is being dragged
2 ondragend script Executed when a drag operation ends
3 ondragenter script Executed when a dragged element enters a valid drop target
4 ondragleave script Executed when a dragged element leaves a valid drop target
5 ondragover script Executed when a dragged element is over a valid drop target
6 ondragstart script Executed when the user starts dragging an element
7 ondrop script Executed when a dragged element is dropped on a valid target
8 onscroll script Executed when an element's scroll bar is being scrolled

Clipboard Event Attributes

1 oncopy script Executed when text or content is copied
2 oncut script Executed when text or content is cut
3 onpaste script Executed when content is pasted into an element

Media Event Attributes

1 onabort script Executed when media loading is aborted
2 oncanplay script Executed when media can start playing without buffering
3 oncanplaythrough script Executed when media can play to the end without buffering
4 oncuechange script Executed when a cue changes in a track element
5 ondurationchange script Executed when the media duration changes
6 onemptied script Executed when media becomes empty (no data)
7 onended script Executed when media playback reaches the end
8 onerror script Executed when an error occurs during media loading or playback
9 onloadeddata script Executed when media data is loaded
10 onloadedmetadata script Executed when metadata (duration, dimensions) is loaded
11 onloadstart script Executed when the browser starts loading media
12 onpause script Executed when media playback is paused
13 onplay script Executed when media playback starts
14 onplaying script Executed when media starts playing after being paused or buffering
15 onprogress script Executed when the browser is downloading media data
16 onratechange script Executed when the playback rate changes
17 onseeked script Executed when seeking has completed
18 onseeking script Executed when seeking begins
19 onstalled script Executed when the browser is trying to fetch media data but is stalled
20 onsuspend script Executed when media loading is intentionally stopped before completion
21 ontimeupdate script Executed when the media's current playback position changes
22 onvolumechange script Executed when the volume or mute state changes
23 onwaiting script Executed when playback is waiting for more data (buffering)
