Selectors

No. Selector Description HTML Example CSS Example
1 ID Selector Selects an element with a specific ID <div id="myDiv">Hello</div> #myDiv { color: red; }
2 Class Selector Selects elements with a specific class <div class="myClass">Hello</div> .myClass { font-size: 20px; }
3 Attribute Selector Selects elements with a specific attribute <input type="text"> [type="text"] { border: 2px solid blue; }
4 Pseudo-class Selector Selects elements based on their state or behavior <a href="#">Link</a> a:hover { color: green; }
5 Child Combinator Selects immediate child elements <div><p>Hello</p></div> div > p { color: blue; }
6 Descendant Combinator Selects all descendant elements <div><span>Hi</span></div> div span { font-weight: bold; }
7 General Sibling Combinator Selects all siblings after a given element <h1>Title</h1><p>Para</p> h1 ~ p { color: purple; }
8 Adjacent Sibling Combinator Selects the element immediately after a given element <h1>Title</h1><p>Para</p> h1 + p { font-style: italic; }
9 Not Combinator Selects elements that do not match a specific selector <div>Hello</div><p>World</p> div:not(.myClass) { background-color: yellow; }
10 nth-child Pseudo-class Selects elements based on their position in a parent element <ul><li>Item 1</li><li>Item 2</li><li>Item 3</li></ul> ul li:nth-child(2) { color: blue; }
11 first-child Pseudo-class Selects the first child element of a parent element <ul><li>Item 1</li><li>Item 2</li><li>Item 3</li></ul> ul li:first-child { font-weight: bold; }
12 last-child Pseudo-class Selects the last child element of a parent element <ul><li>Item 1</li><li>Item 2</li><li>Item 3</li></ul> ul li:last-child { text-decoration: underline; }
13 nth-of-type Pseudo-class Selects elements based on their position in a parent element <ul><li>Item 1</li><li>Item 2</li><li>Item 3</li></ul> ul li:nth-of-type(2) { color: green; }
14 first-of-type Pseudo-class Selects the first child element of a parent element <ul><li>Item 1</li><li>Item 2</li><li>Item 3</li></ul> ul li:first-of-type { font-style: italic; }
15 last-of-type Pseudo-class Selects the last child element of a parent element <ul><li>Item 1</li><li>Item 2</li><li>Item 3</li></ul> ul li:last-of-type { text-decoration: underline; }
16 only-child Pseudo-class Selects elements that have no siblings <div>Hello</div><p>World</p> div:only-child { background-color: yellow; }
17 only-of-type Pseudo-class Selects elements that have no siblings <div>Hello</div><p>World</p> div:only-of-type { background-color: yellow; }
18 empty Pseudo-class Selects elements that have no content <div>Hello</div><p>World</p> div:empty { background-color: yellow; }
19 link Pseudo-class Selects elements that are links <a href="#">Link</a> a:link { color: blue; }
20 visited Pseudo-class Selects elements that have already been visited <a href="#">Link</a> a:visited { color: green; }
21 hover Pseudo-class Selects elements when the user hovers over them <a href="#">Link</a> a:hover { color: green; }
22 active Pseudo-class Selects elements that are currently active <a href="#">Link</a> a:active { color: red; }
23 focus Pseudo-class Selects elements that have focus <input type="text"> input:focus { background-color: yellow; }
24 target Pseudo-class Selects elements that are the target of a link <a href="#section1">Section 1</a> a:target { background-color: yellow; }
25 nth-last-child Pseudo-class Selects elements based on their position in a parent element <ul><li>Item 1</li><li>Item 2</li><li>Item 3</li></ul> ul li:nth-last-child(2) { color: green; }
26 nth-last-of-type Pseudo-class Selects elements based on their position in a parent element <ul><li>Item 1</li><li>Item 2</li><li>Item 3</li></ul> ul li:nth-last-of-type(2) { color: green; }
27 Group Selector Selects multiple elements and applies the same styles to them <h1>Title</h1> <p>Paragraph</p> h1, p { color: red; }
28 All Selector Selects all elements on the page <h1>Title</h1> <p>Paragraph</p> * { color: red; }
